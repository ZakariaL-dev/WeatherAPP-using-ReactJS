## 🌟 Overview
The **React Weather App** is a modern web application that provides users with instant weather updates for cities worldwide. Built with **React.js**, this project demonstrates essential frontend skills including state management, asynchronous API fetching, and component-based architecture.

## ✨ Key Features
* **🌍 Global Search:** Get accurate weather data for any city, state, or country.
* **📊 Live Metrics:** Displays temperature, humidity, wind speed, and weather descriptions.
* **⚡ Instant Updates:** Real-time data fetching using the OpenWeatherMap API.
* **📱 Responsive Layout:** A clean, mobile-friendly interface that adapts to all screen sizes.
* **🎨 Dynamic Icons:** Visual weather indicators (sunny, cloudy, rainy, etc.) based on current conditions.

## 🚀 Installation & Setup

1.  **Clone the Repository**
    ```bash
    git clone [https://github.com/ZakariaL-dev/WeatherAPP-using-ReactJS.git](https://github.com/ZakariaL-dev/WeatherAPP-using-ReactJS.git)
    ```
2.  **Navigate to the Project Directory**
    ```bash
    cd WeatherAPP-using-ReactJS
    ```
3.  **Install Dependencies**
    ```bash
    npm install
    ```
4.  **Configuration**
    Create a `.env` file in the root directory and add your OpenWeatherMap API key:
    ```env
    REACT_APP_WEATHER_API_KEY=your_api_key_here
    ```
5.  **Start the Development Server**
    ```bash
    npm start
    ```

## 📖 How to Use
1.  Enter the name of a city in the search bar.
2.  Press **Enter** or click the **Search** icon.
3.  View the detailed weather statistics and current conditions for that location.
